### 换电脑以及发布新文章

1. 安装 `npm install hexo-cli -g`
2. 克隆代码 `git clone git@....`
3. 进入克隆的文件夹 `npm install`,`npm install hexo-deployer-git --save`
4. 生成`hexo g`
5. 部署 `hexo d`
6. 新建页面`hexo new newPage`
7. `hexo clean` 清除缓存文件和生成的静态文件（当发现对站点的更改不生效时，运行此命令）
<!--more -->

#### 本地运行

`hexo serve`
